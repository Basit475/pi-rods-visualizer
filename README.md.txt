# π Rods Visualizer

A fun Swing app that visualizes π using two rotating rods.

## Features
- Rod B can be locked to rotate at π times Rod A.
- Trails, grid/axis, and phase plot visualizations.
- Start/pause with double-click, toggle info overlay with right-click.
- Export PNG images or time-lapse sequences.
- Presets for π, 22/7, e, or random irrational ratios.

## How to Run
```bash
javac PiRodsVisualizer.java
java PiRodsVisualizer


Controls

Adjust lengths and angular speeds using sliders/spinners.

Toggle lock, trail, and grid with checkboxes.

Use presets to explore interesting patterns.
